# Jeu-en-C
 Description
Ce projet est une implémentation du jeu de Blackjack réalisée en langage C dans le cadre de ma formation au CNAM Paris. Il m'a permis de valider les concepts de programmation structurée et de gestion d'algorithmes complexes.

Fonctionnalités
Moteur de jeu : Gestion complète des tours (Joueur et Banque).

Calcul des scores : Logique de comptage des points incluant la gestion de l'As.

Interface Terminal : Interaction fluide avec l'utilisateur via la console.

Compétences validées au CNAM
Maîtrise de la syntaxe du langage C.

Gestion des structures de contrôle (boucles, conditions).

Organisation du code en fonctions pour une meilleure maintenance.
